### 如何导入模块

1.  **打开 Shadowrocket 应用**：在您的 iOS 设备上启动 Shadowrocket。
2.  **进入配置页面**：点击底部导航栏的“配置”图标。
3.  **添加远程文件**：点击右上角的“+”按钮，选择“添加远程文件”。
4.  **输入 URL**：在弹出的输入框中粘贴 URL。
5.  **保存并更新**：点击“完成”或“保存”，Shadowrocket 将会自动下载并应用该模块。
